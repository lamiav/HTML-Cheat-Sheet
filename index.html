CSS SELECTORS

Selector Types
Selector Description Example Code
HTML CSS
Type Selectors that are direct html tags (ex: p, h1, div etc...) <p> This text is blue. </p> p {
color: blue;
}
Universal One selector for CSS rules applying to all html <h1> This heading is in Helvetica </h1>
<p> This paragraph is written in Helvetica </p>
<span> This span is also in Helvetica </span>		 * {
font-family: Helvetica, sans-serif;
}
Class CSS rules applies to element(s) with corresponding class <h1> This heading is a heading. </h1>
<p class="special"> This paragraph should be in bold </p>		 .special {
font-weight: bold;
}
ID CSS rules applies to element(s) with corresponding id (use sparingly!) <div id="red-section">
<p> All child elements of this div should be red. </p>
</div>		 #red-section {
color: red;
}
Attributes CSS rules applies to specific attributes of html tags or certain qualities of tag attribute

example: <a> elements with a title attribute		 <a> This is a link </a>
<a title="sign-in"> This is another link </a>		 a[title] {
font-weight: bold;
}
example: <a> elements with an href matching "https://example.org" <a> This link will be normal. </a>
<a href="https://example.org"> This link will be pink. </a>		 a[href="https://example.org"] {
background-color: pink;
}
example: <a> elements with an href containing "example" <a href="https://example.org/example1"> This link will be pink. </a>
<a href="https://example.org"> This link will be pink too. </a>		 a[href*="example"] {
background-color: pink;
}
example: <a> elements with an href ending ".org" <a> This link will be normal. </a>
<a href="https://example.org"> This link will be pink. </a>		 a[href$=".org"] {
background-color: pink;
}
example: <a> elements whose class attribute contains the word "logo" <a class="awesome-logo"> This link will be italic. </a>
<a class="awesome"> This link will not. </a>		 a[class~="logo"] {
font-style: italic;
}
Pseudo class Seclector pertaining to elements after an user interaction <p> This text will change size when the cursor is on it. </p> p:hover {
font-size: 30px;
}
Selector Combinations
Combination Description Example Code
HTML CSS
Chaining Selects elements with a specific class <p class="blue"> This text is blue. </p>
<p> This text is not. </p>		 p.blue {
color: blue;
}
Descendant Combinator Selects child elements nested in a parent with a certain class/id <ul class="special-list">
<li>Bold List item</li>
<li>Bold List item</li>
</ul>		 .special-list li {
font-weight: bold;
}
Multiple Selectors Combines multiples elements for one CSS rule <p> All paragraphs are blue. </p>
<h1> All headings are blue as well. </h1>		 p,
h1 {
color: blue;
}
